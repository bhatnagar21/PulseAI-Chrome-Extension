🚀 How to Install
Clone the repo:
git clone <YOUR_REPO_URL>
Open Chrome → chrome://extensions/
Enable Developer Mode
Click Load Unpacked → select pulseAI-extension/
Click PulseAI icon → start timer & add tasks
